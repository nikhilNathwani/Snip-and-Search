Clip & Search — macOS Shortcut

Quickly take a screenshot of any part of your Mac’s screen and instantly search it on Google Images. This shortcut captures a region you select, opens Google Images in Safari, and uploads the screenshot for a reverse image search.

How to install

  1. Click the button below to open the shortcut in the Shortcuts app:
    Install Clip & Search
  2. On first run, macOS will prompt you to allow permissions for screen capture and automation — approve them.

How to use

  • Run the shortcut from the Shortcuts app or assign it a keyboard shortcut.
  • When the crosshair appears, select the part of your screen you want to search.
  • The shortcut will open Safari, navigate to Google Images, and upload the screenshot automatically.

Requirements

  • macOS 12 Monterey or later.
  • Safari browser (automation only works with Safari).

Notes

  • The shortcut uses AppleScript and system automation — allow the necessary permissions in System Settings → Privacy & Security.
  • If the screenshot is too small, the shortcut will cancel automatically to avoid unintentional triggers.
  • This shortcut only works on macOS, not iPhone or iPad.
