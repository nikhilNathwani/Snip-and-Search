Snip & Search

 MacOS shortcut


Take a screenshot on your Mac and automatically search it on Google Images.

Install here

See it in action

In this example: looking at a standing desk → want to identify the lamp → snip just the lamp → automatic image search on Google Images

How to use it

  1. Run the shortcut from the Shortcuts app.
    • If prompted, approve permissions for screen capture and automation.
  2. Click and drag to select the part of your screen you want to search.
  3. Sit back and watch the shortcut automatically:
    • Launch Google Images
    • Perform an image search on your screenshot

How to install

Open the shortcut here, then click "Get Shortcut" to add it to your Shortcuts app.

Requirements

  • macOS 12 Monterey or later
  • Safari browser

Notes

  • For quicker access, assign a keyboard shortcut to run it.
    • I use Command + Shift + 2
  • The shortcut temporarily saves the screenshot to your device, then deletes it after use.
  • If the screenshot is too small, the shortcut will cancel automatically to avoid unintentional triggers.
